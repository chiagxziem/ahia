# Ahia Web

Storefront and Admin Dash for Ahia eCommerce.

- Built with Nextjs.
- Deployed to Vercel. Can also be deployed using the Dockerfile.

## Getting Started

1. Copy `.env.example` to `.env` and configure environment variables.
2. Run `bun install` to install dependencies.
3. Run `turbo dev` from the project root to start the app.

See the main project [`README.md`](https://github.com/gozmanthefirst/ahia) for more detailed.
